📊 典型事务流程

✏️ Memory Write (D2H DMA)

1
User writes DMA descriptor via AXI4-Lite
2
DMA Engine reads local memory (AXI4)
3
DMA generates MWr TlpStreamPacket
4
TLP TX Engine serializes to 32-bit DWORDs
5
DL Framer adds SeqNum + LCRC
6
PHY 8b/10b encodes → SerDes TX
7
Root Complex ACKs → D2H Done flag

📖 Memory Read (H2D DMA)

1
DMA issues MRd TLP (tag assigned)
2
TLP TX → DL → PHY → Root Complex
3
Root Complex returns CplD TLP
4
PHY RX → DL deframes + verifies LCRC
5
TLP RX Engine parses, matches tag
6
Completion data sent to DMA Engine
7
DMA writes data to local AXI memory

⚙️ Config Read (Enumeration)

1
Root Complex sends CfgRd0 TLP
2
RX Engine routes to Config Space Ctrl
3
Decode DWORD address (0x00=VID/DID…)
4
Config Space generates CplD TLP
5
TX Engine → DL → PHY → Root Complex

🔔 MSI-X Interrupt

1
User asserts intReq[n]
2
MSI-X Ctrl scans pending bits
3
Read vector address/data from MSI-X table
4
Generate Msg TLP (MsgD format)
5
TX Engine → DL → PHY → Root Complex
6
intAck[n] asserted, pending bit cleared
